🎬 Movies App

A Flutter-based mobile application that allows users to explore trending movies, search for films, and view detailed information such as ratings, synopsis, trailers, and more.

📱 Features
🔍 Search for movies by name
📈 View trending and popular movies
🎬 Detailed movie page (overview, rating, release date)
⭐ User-friendly UI with smooth navigation
🌐 API integration for real-time movie data
❤️ Favorite/save movies (if implemented)
🛠️ Tech Stack
Flutter (UI Framework)
Dart (Programming Language)
REST API (Movie database integration)
Provider / Riverpod / Bloc (state management – update based on yours)
HTTP / Dio (network calls)
