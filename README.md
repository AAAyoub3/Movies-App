# 🎬 Movies App

A sleek Flutter application that allows users to explore trending movies, search for titles, and view detailed information including ratings, summaries, and more — powered by real-time movie data APIs.

---

## ✨ Features

- 🔍 Search movies instantly by title  
- 📈 Discover trending & popular movies  
- 🎬 Detailed movie information (overview, rating, release date)  
- 🎥 Clean and modern UI with smooth navigation  
- 🌐 Real-time API integration  
- ❤️ Favorites system (if implemented)

---

## 🛠️ Tech Stack

- **Flutter** – UI framework  
- **Dart** – Programming language  
- **REST API** – Movie data source (e.g. TMDB)  
- **State Management** – (Bloc / Provider / Riverpod)  
- **HTTP / Dio** – Network requests
